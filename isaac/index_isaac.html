Introducción

La presente página tiene como objetivo mostrar cómo se desarrollaron distintas páginas, incluyendo la mía, utilizando diferentes métodos basados en HTML5 y CSS3. Es sorprendente todo lo que se puede llegar a hacer con estas dos bases, pero es aún más impresionante el hecho de que si expandimos nuestros horizontes e implementamos otros tipos de lenguaje, se pueden hacer mil y una cosas más. Los lenguajes que permiten esto son principalmente JavaScript, Bootstrap, PHP, entre otros.

HTML5

MARCADO DE HIPERTEXTO

HTML5 es la última versión del lenguaje de marcado de hipertexto (HTML), que se utiliza para crear páginas web.

CSS3

ESTILO Y DISEÑO

CSS3 es la última versión del lenguaje de hojas de estilo en cascada (CSS), que se utiliza para dar estilo y diseño a páginas web creadas con HTML.

JavaScript

LENGUAJE DE ALTO NIVEL

JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos, que se utiliza principalmente para programar interacciones dinámicas en páginas web.
